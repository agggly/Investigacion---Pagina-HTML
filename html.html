Explicación de Elementos HTML

A continuación, exploraremos algunos de los elementos fundamentales que componen una página web.

<!DOCTYPE html>

Esta declaración le indica al navegador qué versión de HTML se está utilizando. Para HTML5, es simplemente <!DOCTYPE html>.

<html lang="es">

Es el elemento raíz de todas las páginas HTML. El atributo lang="es" especifica el idioma del contenido de la página, en este caso, español.

<head>

Contiene metadatos sobre la página que no son visibles directamente en el navegador, como el título de la página, los enlaces a hojas de estilo, y las codificaciones de caracteres.

<meta charset="UTF-8">

Define la codificación de caracteres para el documento. UTF-8 es una codificación universal que soporta una amplia gama de caracteres.

<title>

Define el título de la página web, que aparece en la pestaña del navegador o en la barra de título de la ventana. Es crucial para la usabilidad y el SEO.

<body>

Contiene todo el contenido visible de la página web, como textos, imágenes, enlaces, videos, etc.

<h1> a <h6>

Son los elementos de encabezado. <h1> es el título más importante y <h6> el menos importante. Solo debe haber un <h1> por página por razones de semántica y SEO.

<p>

Define un párrafo de texto. Es el elemento más común para bloques de texto.

<a href="...">

Crea un hipervínculo que enlaza a otra página web o a una sección dentro de la misma página. El atributo href especifica la URL de destino.

<footer>

Representa el pie de página de un documento o sección. Típicamente contiene información sobre el autor, derechos de autor, enlaces a documentos relacionados, etc.
