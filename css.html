Explicación de CSS

Aquí aprenderemos sobre **CSS**, que nos ayuda a que las páginas web se vean bonitas. Piensa en CSS como la "ropa" de tu página HTML.

<link rel="stylesheet" href="style.css">

Con esta línea en la sección <head>, le decimos a HTML que use los estilos de un archivo llamado style.css. Así, HTML sabe dónde encontrar su "ropa".

`body {}`

El selector **`body`** aplica estilos a **toda la parte visible** de tu página web. Como puedes ver, esta página tiene un color de fondo suave y un tipo de letra básico gracias a estos estilos.

`font-family`

Esta propiedad decide el **tipo de letra** de tu texto. Mira este subtítulo; ¡tiene una fuente diferente para mostrarte el efecto!

`margin` y `padding`

Este es un cuadro

Piensa en tus elementos HTML como cajas. El **`padding`** es el espacio *dentro* de la caja (entre el texto y el borde), y el **`margin`** es el espacio *fuera* de la caja (entre esta caja y otras cajas). Mira el cuadro de ejemplo para entenderlo mejor.

`background-color`

Con esta propiedad, puedes ponerle un **color de fondo** a cualquier parte de tu página. Este subtítulo tiene un color de fondo ligero para que veas cómo funciona.

`color`

Esta es simple: **`color`** cambia el **color del texto**. Mira el texto de este subtítulo, ¡es de un color llamativo!

`font-size`

¿Quieres que tu texto sea grande o pequeño? Usa **`font-size`** para cambiar el **tamaño de la letra**. Este subtítulo es un buen ejemplo, ¡es más grande de lo normal!

`position: relative;`

Esto es un poco más avanzado, pero **`position: relative;`** nos ayuda a que otros elementos (como los pequeños cuadros de información que aparecen al pasar el mouse) puedan "moverse" en relación con este elemento. Así, el pequeño cuadro blanco siempre aparecerá justo debajo del título al que pertenece.

`::after`

El **`::after`** es como añadir algo extra *después* del contenido de un elemento, sin tener que escribirlo en el HTML. Lo usamos aquí para mostrarte la información de color y fuente al pasar el mouse.

`content`

Esta propiedad se usa junto con **`::after`** para decirle qué texto o contenido debe mostrar. Es lo que hace que aparezca el "Color: #..." en los pequeños cuadros al pasar el mouse.

`opacity`

Con **`opacity`**, puedes hacer que algo sea más o menos **transparente**. Un valor de 0 es completamente invisible y 1 es totalmente sólido. Este subtítulo es un poco transparente para que lo notes.

`transition`

**`transition`** hace que los cambios en los estilos (como la aparición del pequeño cuadro de información) sean **suaves y animados**, en lugar de aparecer de golpe. Intenta pasar el mouse sobre los títulos para ver cómo la información aparece suavemente.
